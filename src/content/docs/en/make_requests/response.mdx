### Example of Response 
```json

  {
    "data": {...}         // Data of response
    "success": true,      // Flag indicating whether the request was resolved successfully
    "details": {          // response metadata
      "errors": [],       // List of Errors
      "message": "...",   // Simple Message of response
      "code": "1200",     // Code App of response
      "trace_id": "...",  // UUID by trace the operation
      "caller_id": "..."  // ID of App Caller
    }
  }
```
