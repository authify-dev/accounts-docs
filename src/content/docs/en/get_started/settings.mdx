# Configuration Variables

Below are the configuration variables of the `Settings` class, organized by category.

---

## General Project Settings

| **Variable Name**                   | **Description**                                   | **Type and Default Value**    |
|-------------------------------------|--------------------------------------------------|--------------------------------|
| `ENVIRONMENT`                       | Deployment environment                           | `str`                         |
| `PROJECT_ID`                        | Unique project identifier                        | `str="API0002"`               |
| `TIME_ZONE`                         | Default time zone                                | `str="utc"`                   |
| `TIME_ZONE_UTC`                     | UTC time zone                                    | `str="utc"`                   |
| `DATE_FORMAT`                       | Date format                                      | `str="%Y-%m-%d"`              |
| `DATE_TIME_FORMAT`                  | Date and time format                             | `str="%Y-%m-%d %H:%M:%S"`     |
| `CORS_ORIGINS`                      | Allowed origins for CORS requests                | `list[str]=["*"]`             |
| `SHOW_CRUDS_IN_SWAGGER_SCHEMA`      | Show or hide CRUD operations in Swagger          | `bool=False`                  |

---

## Password Settings

| **Variable Name**                   | **Description**                                   | **Type and Default Value**    |
|-------------------------------------|--------------------------------------------------|--------------------------------|
| `HASHING_ALGORITHM`                 | Hashing algorithm for passwords                  | `HashingAlgorithmsEnum=BCRYPT`|
| `PASSWORD_MIN_LENGTH`               | Minimum password length                          | `int=8`                       |
| `PASSWORD_MAX_LENGTH`               | Maximum password length                          | `int=20`                      |
| `PASSWORD_UPPERCASE_COUNT`          | Minimum number of uppercase characters           | `int=1`                       |
| `PASSWORD_LOWERCASE_COUNT`          | Minimum number of lowercase characters           | `int=1`                       |
| `PASSWORD_DIGIT_COUNT`              | Minimum number of digits                         | `int=1`                       |
| `PASSWORD_SPECIAL_CHAR_COUNT`       | Minimum number of special characters             | `int=1`                       |
| `PASSWORD_SPECIAL_CHARS`            | Allowed special characters                       | `str="[!@#$%^&*(),.?\":{}|\<>]"`|
| `PASSWORD_VALIDATIONS`              | Password validation rules                        | `list[str]=[...]`             |
